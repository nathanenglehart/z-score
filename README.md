# Standard Score
## Author
Nathan Englehart (Summer, 2022)
## Usage
To run this program on any unix-based system:
```bash
git clone https://github.com/nathanenglehart/z-score
cd z-score
gcc -lm z-score.c -o z-score
./z-score
```
